Overall Rating Strong Accept
Detailed Comments This paper describes FactWatcher, which is a system for finding interesting facts in multidimensional databases, which can lead to news stories. FactWatcher includes algorithms for finding interesting situational facts, one-of-the-few facts and prominent streaks, which have been published in KDD, ICDE and TKDD. This looks like a very interesting system that should make a nice demo.

Question 
Overall Rating Accept
Detailed Comments The FactWatcher system is a system that automatically extracts three kinds of facts (situational, one-of-the-few, prominent streak) from data. This system does look useful for computational journalism as it is an automatic method for monitoring important facts from all the stories.

The demonstration provides a nice overview of the different components of the system. It also presents a nice user interface that is backed by sound algorithms used to monitor each type of fact.
